# Success Metrics

This document defines the success metrics for the HYROX Decision Alignment Platform.

All metrics are anchored to a future competition and a target time.

The product exists to maximize competition readiness with limited training time.

---

## Core Product Objective

Help an amateur HYROX athlete achieve or beat their target competition time through better weekly training decisions.

---

## Metric Categories

Metrics are grouped into:

- Competition Outcome Metrics
- Preparation Quality Metrics
- Weekly Execution Metrics
- Decision Clarity Metrics
- Product Trust Metrics

---

## 1. Competition Outcome Metrics

These are the ultimate success indicators.

| Metric | Description |
|------|------------|
| Target Time Achievement | Athlete finishes at or below target time |
| Time Improvement | Difference between last competition and current |
| Placement Improvement | Category ranking evolution |

Success Example:

Athlete target: 1:05:00  
Athlete result: 1:02:30 → Success

---

## 2. Preparation Quality Metrics

Measures readiness before competition.

| Metric | Description |
|------|------------|
| Competition Readiness Score | Composite score from alignment history |
| Volume Completion Rate | % of weekly volume targets met |
| Misalignment Weeks | Number of poorly aligned weeks |

---

## 3. Weekly Execution Metrics

Measures discipline and consistency.

| Metric | Description |
|------|------------|
| Weekly Volume Adherence | % of target volume completed |
| Session Completion Rate | Planned vs executed sessions |
| Overtraining Rate | Weeks exceeding safe volume |

---

## 4. Decision Clarity Metrics

Measures if athlete understands decisions.

| Metric | Description |
|------|------------|
| Recommendation Acceptance Rate | % of recommendations followed |
| Explanation Engagement | % of explanations read |
| Clarity Feedback Score | Athlete self-reported clarity |

---

## 5. Product Trust Metrics

Measures credibility.

| Metric | Description |
|------|------------|
| Recommendation Trust Score | Athlete rating |
| Return Usage Rate | Weekly active usage |
| Manual Override Rate | Athlete ignoring system guidance |

Low manual override = high trust.

---

## 6. Alignment Metrics

| Metric | Description |
|------|------------|
| Weekly Alignment Score | 0–100 |
| Alignment Trend | Week-over-week evolution |
| Alignment Stability | Variance over time |
